# Logo Images Directory

Place your logo images here:

1. **logo1.png** - Municipal/Provincial Seal

   - Recommended size: 80x80 pixels (PNG with transparent background)

2. **logo2.png** - Barangay Seal
   - Recommended size: 80x80 pixels (PNG with transparent background)

These logos will appear on:

- Login page header
- Dashboard header
- Printed reports

## Image Requirements

- Format: PNG (preferred for transparency) or JPG
- Size: 80x80 pixels to 150x150 pixels
- Background: Transparent (for PNG)
- Quality: High resolution for print

## Fallback

If images are not found, they will be hidden automatically using CSS.
